# Phantom Card Battle v0.1.0

PWA + JavaScript + Canvas / PixiJS で作成した、3×3カードバトルの初期版です。

## 内容

- タイトル画面
- 対戦画面
  - 3×3盤面
  - 自分の手札5枚
  - 相手の手札5枚は裏向き表示
  - 現在ターン、スコア、ログ表示
  - カード配置・反転エフェクト
- デッキ画面
  - デッキ1〜5保存
  - ★5は1枚まで
  - ★4は2枚まで
  - ★3以下は無制限
- 図鑑
  - 取得済みカード表示
  - 未取得カードは「???」表示
- 設定
  - エフェクトON/OFF
  - セーブデータのエクスポート/インポート
- PWA
  - manifest
  - service worker
  - バージョン表示
  - 最新版に更新ボタン

## 起動方法

ローカルで確認する場合は、フォルダ内で下記のように簡易サーバーを起動してください。

```bash
python -m http.server 8000
```

その後、ブラウザで開きます。

```text
http://localhost:8000/
```

## カードデータについて

`Phantom_Card_Battle.csv` から `src/data/cards.js` を生成しています。

CSV内の `A` はカード値 `10` として扱っています。  
`*1` などの特殊カード番号は `card_sp_001` のようなIDに変換しています。

## 公開時の注意

添付CSV内に既存作品由来と思われる名称が含まれている場合、公開前にカード名・絵柄・BGM・効果音などはオリジナル化してください。
